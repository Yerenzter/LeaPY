# LeaPY
Is a Python3 library which determines whether a year is a leap year or not.

## Documentation

This is a function to calculate a leap year.
```
leap(year)
```

## Usage
```
import leap

year = [2000, 2001, 2002, 2003, 2004, 2010, 2012, 2016, 2020, 2022, 2024]

for years in year:
    print(leap(years))
```
